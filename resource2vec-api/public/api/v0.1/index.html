    Get or compute knowledge graph embeddings for a given RDF dataset.

    HTTP Request

    POST http://resource2vec.aksw.org/process

    Query Parameters

    Parameter Default Description
    name - The dataset name. In case file or dataset are not null, it will be applied as label to the newly loaded dataset.
    file Optional An RDF file in any format.
    dataset Optional The dataset URL. If file is not null, it will be ignored.
    method - The knowledge graph embedding method.
    hyperp - A map of hyperparameters as string (e.g., par1=123;par2=456).

    Errors

    The Resource2Vec API uses the following error codes:

    Error Code Meaning
    400 Bad Request
    401 Unauthorized
    403 Forbidden
    404 Not Found
    405 Method Not Allowed
    406 Not Acceptable
    410 Gone
    418 I’m a teapot
    429 Too Many Requests
    500 Internal Server Error
    503 Service Unavailable
